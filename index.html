SPACE & BEYOND

Quality Assurance Testing on SPACE & BEYOND

Introduction

Testing is a very important part of software development. It ensures that applications function correctly and meet user expectations. Thanks to testing, we can identify errors and improve the quality of the final product.
With this in mind, I created this website to showcase the work I did in the QA Testing course at Coderhouse. Creating this page helps me review what I learned in class about testing and continue practicing my web development skills. I believe there is no better way to apply what I've learned than through this project.
